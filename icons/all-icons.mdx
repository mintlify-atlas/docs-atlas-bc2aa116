## Naming Convention

All icon names follow a consistent kebab-case pattern:

```tsx
import { HeartIcon } from '@/components/ui/heart';
import { ArrowRightIcon } from '@/components/ui/arrow-right';
import { CloudDownloadIcon } from '@/components/ui/cloud-download';
```

**Pattern:**
- Icon names are in PascalCase with `Icon` suffix
- Source files use kebab-case (e.g., `heart`, `arrow-right`, `cloud-download`)
- Component names transform kebab-case to PascalCase + Icon

## Finding Icons

### By Name
If you know roughly what you're looking for, icon names are intuitive:
- Directional: `arrow-*`, `chevron-*`, `corner-*`
- Actions: `download`, `upload`, `send`, `delete`, `copy`
- States: `*-check`, `*-off`, `*-plus`, `*-minus`

### By Keyword
Every icon has multiple keywords. For example:
- **heart**: `like`, `love`, `emotion`
- **search**: `find`, `scan`, `magnifier`
- **logout**: `sign out`, `exit`, `auth`
- **trash**: `garbage`, `delete`, `remove`, `bin`
